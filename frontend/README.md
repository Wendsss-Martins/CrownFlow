# CrownFlow

CrownFlow é uma plataforma SaaS desenvolvida para ajudar barbeiros autônomos e pequenas barbearias a organizar agendamentos, registrar serviços e acompanhar o desempenho do negócio. O sistema reúne agenda, gestão básica e dashboards financeiros em uma interface simples e intuitiva.

## 🚀 Funcionalidades

* 📅 Sistema de agendamento de clientes
* 💈 Registro de serviços realizados
* 📊 Dashboards com indicadores de faturamento
* 📈 Relatórios de desempenho
* 📱 Integração com WhatsApp para comunicação com clientes
* 🧑‍💼 Gestão de barbeiros e serviços

## 🛠️ Tecnologias utilizadas

* React
* JavaScript
* TailwindCSS
* Radix UI
* Axios
* Recharts

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/crownflow.git
```

Acesse a pasta do projeto:

```bash
cd crownflow/frontend
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm start
```

A aplicação será iniciada em:

```
http://localhost:3000
```

## 🎯 Objetivo do projeto

O CrownFlow foi criado com o objetivo de digitalizar a gestão de barbeiros iniciantes, oferecendo uma ferramenta simples para controle de agenda, acompanhamento de faturamento e organização do negócio.
